# Pong Game Documentation

## Features
- Classic Pong gameplay
- Single player and multiplayer modes
- Score tracking
- Adjustable paddle speed
- Full-screen mode

## Controls
- **Player 1:**
  - Up Arrow: Move paddle up
  - Down Arrow: Move paddle down

- **Player 2:**
  - W: Move paddle up
  - S: Move paddle down

## Instructions
1. Start the game by launching the application.
2. Choose a mode: Single Player or Multiplayer.
3. Use the controls to move the paddles and score points.
4. The game ends when a player reaches the maximum score.

Enjoy the game!
